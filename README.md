# Twin Prime Power Law

Computational scripts for "The Hardy--Littlewood Residual for Twin Primes Follows a Power Law."

## Files

- `count_twins_10_10.py` -- segmented sieve, pi_2(10^10)
- `prit_powerlaw.py` -- growth ratios, log-log fit, extrapolation
- `prit_val4.py` -- 4-point model validation
- `prit_final.py` -- oscillatory fit (negative result)

## Requirements

    numpy, scipy

## Data

pi_2(10^k) from Nicely's tables. C_2 = 0.6601618914. First 50 zeta zeros.
